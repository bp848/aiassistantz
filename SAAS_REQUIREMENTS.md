# クラウド社長室Z SaaS化要件

## 🏢 SaaSモデル
- **B2B向け**: 企業向けAI秘書SaaS
- **月額課金**: 1社長あたり月額制
- **マルチテナント**: 各企業が独立環境
- **プラン体系**: Basic → Pro → Enterprise

## 🎯 必要機能

### 1. テナント管理
- [ ] 企業登録・管理
- [ ] ユーザー招待・権限管理
- [ ] 請求プラン選択・課金
- [ ] テナントごとのデータ分離

### 2. 認証・認可
- [ ] 企業単位のSSO対応
- [ ] Google Workspace連携（テナントごと）
- [ ] ロールベース権限管理
- [ ] APIキー管理

### 3. データ管理
- [ ] テナントごとのデータ分離
- [ ] データバックアップ・復元
- [ ] データポリシー管理
- [ ] コンプライアンス対応

### 4. 課金システム
- [ ] プラン管理（Basic/Pro/Enterprise）
- [ ] 月額請求・支払い
- [ ] 利用量監視・制限
- [ ] 請求書・領収書発行

### 5. 管理機能
- [ ] テナント管理ダッシュボード
- [ ] 利用状況レポート
- [ ] サポート・問い合わせ管理
- [ ] システム設定

## 💰 プラン体系

### Basicプラン（月額5,000円）
- AI秘書基本機能
- Google Calendar連携
- Gmail連携（1アカウント）
- 基本的なデータ管理
- メールサポート

### Proプラン（月額15,000円）
- Basicプラン全機能
- 複数Googleアカウント連携
- 高度なAI機能
- カスタマイズ可能
- チ中サポート

### Enterpriseプラン（月額50,000円〜）
- Proプラン全機能
- 無制限ユーザー数
- 専用サーバー
- APIアクセス
- 儡制限サポート
- カスタム開発

## 🔧 技術要件

### マルチテナント対応
```typescript
interface Tenant {
  id: string;
  name: string;
  plan: 'basic' | 'pro' | 'enterprise';
  domain: string;
  settings: TenantSettings;
  users: User[];
  createdAt: Date;
}

interface User {
  id: string;
  tenantId: string;
  email: string;
  role: 'admin' | 'user';
  profile: UserProfile;
}
```

### データ分離
- Supabase Row Level Security (RLS)
- テナントIDベースのデータ分離
- APIキーのテナントごと管理

### 認証拡張
- Supabase Auth + テナント管理
- Google Workspace SSO
- JWTトークンにテナント情報を含める
